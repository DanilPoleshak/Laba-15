Данная работа представляет десктопную систему управления складом и доставкой, использованием Windows Forms и SQLite.
Она включает модуль склада для учёта товаров (с ограничениями по цене и количеству), импортом из CSV и корректным управлением ID без пропусков
Имеется модуль доставки для оформления заказов с автоматическим списанием товаров со склада.
Все данные хранятся в локальной базе данных на рабочем столе, обеспечивается защита от дубликатов и удобный пользовательский интерфейс с цветовой индикацией и валидацией ввода.
